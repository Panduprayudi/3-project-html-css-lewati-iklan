Cara download Melewati Via   -   adf.ly

  1. Klik shortcut tujuan download.




  2. Setelah muncul halaman adf.ly (seperti gambar), tunggu sampai 5 detik.



  3. Setelah 5 detik, maka klik SKIP AD pada pojok kanan atas (seperti gambar).

  4. Apabila sudah klik SKIP AD maka halaman download yag asli akan muncul, atau biasanya langsung download filenya.



Cara download Melewati Via   -   Safelink

  1. Klik shortcut tujuan download.




  2. Setelah tunggu 10 detik dan centang, selanjutnya klik tombol “Click here to continue”.




  3. Setelah klik “Click here to continue” berikutnya tunggu hingga muncul tombol “Get Link”.




  4. Klik tombol “Get Link” untuk menuju ke link download.

Cara Extract File Part

  1. Pastikan Semua file nama-nya harus sama.




  2. Jangan sampai ini terjadi yaa, pastikan nama file part-nya harus sama semuanya.




  3. Cara Mengatasinya, Cukup rename nama-nya saja.




  4. Terkhir jika nama file sudah sama semuanya lalu di extract dan muncul error, coba download ulang part yang corrupt.

  5. dan coba di extract, Selesai.
